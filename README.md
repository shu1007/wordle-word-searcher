# wordle-word-searcher

wordle の攻略を助けるツールです

## 使い方

-   説明に沿って入力を行っていく

    -   Enter the green letters
        -   緑の文字を入力する
        -   不明な箇所は.を入力
        -   e.g. w....
    -   Enter the yellow letters
        -   黄色の文字を入力する
        -   それぞれをそのまま続けて入力
        -   e.g. abc
    -   Enter the black letters
        -   黒の文字を入力する
        -   それぞれをそのまま続けて入力
        -   e.g. abc

-   条件に合致する単語が出力される
